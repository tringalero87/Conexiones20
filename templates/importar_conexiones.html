Instrucciones

Para asegurar una importación exitosa, tu archivo de Excel debe seguir el siguiente formato:

Columnas Requeridas:
  • TIPO: El tipo principal de conexión (Ej: MOMENTO, CORTANTE).
  • SUBTIPO: El subtipo de la conexión (Ej: VIGA-COLUMNA (ALA)).
  • TIPOLOGIA: El nombre de la tipología (Ej: T1, T2).
  • PERFIL1: El valor para el primer perfil.
  • PERFIL2: El valor para el segundo perfil (opcional, dejar en blanco si no aplica).
  • DESCRIPCION: Una descripción opcional para la conexión.

¿No estás seguro del formato? Descarga nuestra plantilla de ejemplo.
