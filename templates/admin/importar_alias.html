Instrucciones

Tu archivo debe contener las siguientes columnas exactas para una importación exitosa:

Columnas Requeridas:
  • NOMBRE_PERFIL: Nombre completo del perfil (Ej: IPE 200, HEA 240).
  • ALIAS: El alias corto deseado para el perfil (Ej: P200, H240).
  • NORMA: (Opcional) La norma bajo la cual se rige el perfil (Ej: AISC, Eurocode).

Asegúrate de que los nombres de las columnas en tu archivo coincidan exactamente.
